# Swift Career Hub

A content website for Swift, SwiftUI, iOS interview preparation, and LeetCode solutions in Swift.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Import this repository in Vercel and deploy.

## AdSense preparation checklist

- Publish original, helpful articles.
- Add About, Contact, Privacy Policy, Terms, and Disclaimer pages.
- Submit sitemap to Google Search Console.
- Avoid copied or auto-generated low-quality content.
